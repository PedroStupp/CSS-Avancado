Clip-path

O clip-path é uma propriedade CSS que permite criar formas complexas cortando elementos de acordo com um caminho específico. É util para criar layouts e efeitos visuais interessantes.

Como funciona o clip-path

O cli-path define uma área vísivel de um elemento, escondendo o que está fora dessa área. Existem várias formas que voce pode usar com clip-path, como:

  • polygon(): Define uma forma usando uma lista de coodenadas que formam um poligono.
  • circle(): Define uma forma circular.
  • ellipse(): Define uma forma eíptica.
  • inset(): Define um retângulo interno com uma margem opcional.
